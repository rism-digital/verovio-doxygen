Detailed Description

This class is an abstract device context.

It enables different types of concrete classes to be implemented. For example: MusWxDC - a wrapper to wxDCs with wxWidgets; SvgDeviceContext - a non-gui file DC; MusCairoDC - a wrapper to a Cairo surface; The class uses int-based colour encoding (instead of wxColour in wxDC). It uses FontInfo (instead of wxFont in wxDC).
