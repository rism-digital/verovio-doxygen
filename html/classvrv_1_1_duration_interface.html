Detailed Description

This class is an interface for elements with duration, such as notes and rests.

It is not an abstract class but should not be instanciated directly.

Returns the duration (in double) for the element.

It returns 0.0 for grace notes. Careful: this method is not overriding LayerElement::GetAlignmentDuration since LayerElement and DurationInterface have no inheritance link.
