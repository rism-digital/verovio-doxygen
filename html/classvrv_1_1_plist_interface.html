Detailed Description

This class is an interface for elements having a single time point, such as tempo, reh, etc.

It is not an abstract class but should not be instantiated directly.

We have functor in the interface for avoiding code duplication in each implementation class.

Since we are in an interface, we need to pass the Object (implementation) to the functor method. These not called by the Process/Call loop but by the implementaion classes explicitely. See FloatingObject::FillStaffCurrentTimeSpanning for an example. See Object::PreparePlist
