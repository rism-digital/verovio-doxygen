◆ ClassId

enum vrv::ClassId : uint16_t

The ClassIds are used to identify Object child classes through the Object::Is virtual method.

Each Object child class has to have its own id which is stored in the member variable m_classId. Base classes (e.g., LayerElement) that are never instanciated have boundary ids used for checking if an Object is child of a base class. See for example Object::IsLayerElement.
