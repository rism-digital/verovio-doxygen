Since we are in an interface, we need to pass the Object (implementation) to the functor method. These not called by the Process/Call loop but by the implementaion classes explicitely. See FloatingObject::FillStaffCurrentTimeSpanning for an example. See Object::FillStaffCurrentTimeSpanning
